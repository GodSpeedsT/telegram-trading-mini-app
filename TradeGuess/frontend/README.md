# Как запустить frontend

## Скачиваем зависимости

```sh
npm install
```

### Запускаем в dev режиме

```sh
npm run dev
```

## API
Мы использовали API:
 
1. binance: 'https://api.binance.com/api/v3'
2. coincap: 'https://api.coincap.io/v2'
3. coinranking: 'https://api.coinranking.com/v2'
4. fmp: 'https://financialmodelingprep.com/api/v3'
5. alphavantage: 'https://www.alphavantage.co/query'
6. yahooFinance: 'https://query1.finance.yahoo.com/v8/finance/chart'
